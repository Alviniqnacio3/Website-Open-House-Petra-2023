# 🎓 Petra Open House 2023 Web Application

## 📌 Overview

This project is a university-wide web application developed for **Petra Christian University Open House 2023**.
The platform was used by **1,000+ students** to explore, register, and participate in various Student Activity Units and Organizations.

The system serves as:

* An **information hub** for student organizations
* A **registration and management system** for event participation

This project was developed as a **Fullstack Developer & IT Team Coordinator**.

---

## ⚙️ Tech Stack

* PHP (Native)
* JavaScript
* HTML & CSS
* Bootstrap
* MySQL

---

## 🚀 Key Features

### 🏫 1. Organization & Club Management

* Centralized platform to showcase student clubs and activity units
* Each club has its own customizable page:

  * Description
  * Activities
  * Photos & logo
* Fully manageable through admin panel

---

### 🧾 2. Registration System

* Students can register for multiple organizations
* Dynamic registration workflow
* Each organization can define its own registration rules

---

### 💳 3. Payment Integration (QR-Based)

* QR code-based payment system
* Dynamic registration fee per organization
* Payment instructions displayed automatically
